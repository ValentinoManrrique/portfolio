// 01 — PROBLEM

Every lead started with a blank email

Potential clients would reach out with vague requests — "I need a website", "I want an app". Before any scoping could happen, there was always a round of back-and-forth emails just to understand the project type, budget, and timeline.

This added friction on both sides. Clients didn't know what information mattered. I spent time on discovery calls that could have been handled asynchronously.

The goal: let a client go from "I have an idea" to a structured, scoped project brief in under 2 minutes — without any manual input from me.

// 02 — SOLUTION

A 5-question form that thinks for itself

A guided form embedded directly in the portfolio collects 5 inputs: project type, main goal, timeline, budget, and the biggest challenge. On submit, it fires a POST request to an n8n webhook.

n8n passes the data to a Basic LLM Chain node using Claude Haiku — fast, cheap, and accurate for structured JSON output. The AI returns a complete project brief: a project name, scope overview, suggested phases, and clarifying questions.

The brief renders in the browser in real time. No page reload. No email. No waiting.

// 03 — BUILD

How it was built

01

Form UI in vanilla JS

5-step wizard with option buttons and a free-text textarea. Progress bar, back/next navigation, and a summary view after submit. No framework — pure JS IIFE embedded in the portfolio HTML.

02

Animated pipeline visual

While the request runs, the right side shows a live node diagram: Browser → Webhook → Anthropic → Response. Each node lights up as the request passes through it using CSS transitions and a setInterval timer.

03

n8n workflow

Webhook node receives form data → Basic LLM Chain calls Claude Haiku with a tight ~110 token prompt → Code node parses the JSON response → Respond to Webhook sends the brief back to the browser.

04

Fallback brief

If the webhook fails or the AI returns malformed JSON, a structured fallback brief is generated client-side from the form answers — so the user always gets something useful.

// 04 — RESULTS

Measurable outcomes

<2min Time to brief
~$0.001 Cost per brief
0 Manual steps

Every form submission auto-logs to Google Sheets — project type, budget range, challenge, and generated brief name — building a lead database passively.
